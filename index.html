¡Bienvenido al servidor HTTP con Node.js!

Node.js es una tecnología que permite construir servidores web eficientes y escalables.

En este ejemplo, utilizamos el módulo 'http' de Node.js para crear un servidor. Cuando alguien accede al servidor, se envía como respuesta la página 'index.html'.
